- 소속사(MAA) 프로필(커버리지 기준): https://maa.co.kr/artists/go-younjung

## 출처
- docs/editorial_policy.md

> 목표: "배우 고윤정의 모든 것을 담은 백과사전형 위키" 품질 기준을 측정 가능한 항목으로 고정.

## 1) 데이터 모델
- [ ] 공통 엔티티 정의 완료: 인물/작품/배역/수상/브랜드/인터뷰/행사/화보/출처
- [ ] 각 엔티티 최소 필드 정의: `id`, `title`, `date`, `status`, `sources`, `last_verified_at`
- [ ] 상태 태그 체계 통일: `공식확정`, `보도(1차)`, `참고(2차)`, `확인 필요`
- [ ] 출처 등급(S/A/B)와 상태 태그 매핑 규칙 문서화
- [ ] 중복 판정 키 정의: `(정규화 URL + 날짜 + 제목)` 또는 `(공식 ID)`

## 2) 페이지 구조
- [ ] 입구 페이지 계층 명확화: `index.md -> hub.md -> domain pages`
- [ ] 모든 도메인에 연도별 인덱스 존재(works/interviews/appearances/pictorials/endorsements)
- [ ] 작품 페이지 템플릿 준수율 100% (`pages/works/*.md`)
- [ ] 각 작품 페이지에 링크 박스 1개 이상(공식 페이지 포함)
- [ ] KR/EN 허브 간 핵심 링크 동기화

## 3) 커버리지
- [ ] 필모그래피 주요 작품 누락 0건
- [ ] 수상/노미네이트 각 행에 근거 링크 1개 이상
- [ ] 인터뷰 항목 최소 요건 충족: 요약 3~5줄 + 키워드 3개+
- [ ] 출연/행사 페이지 연도 공백 구간 명시 및 백로그 연결
- [ ] 화보(커버/에디토리얼/캠페인/메이킹) 카테고리별 최소 목표치 충족

## 4) 품질 지표
- [ ] placeholder 총량 추세 하락(주간 기준)
- [ ] `교차검증 필요` 항목 감소율 주간 5% 이상
- [ ] `요약 보강 필요` 항목 감소율 주간 5% 이상
- [ ] broken/empty link 0건(`lint` + `link-health`)
- [ ] 표기 규칙 위반(날짜 형식/상태 태그 누락) 0건

## 5) 편집 거버넌스
- [ ] 사생활/루머 금지 규칙 점검 완료
- [ ] 저작권/인용 원칙 준수(원문 복붙 금지, 요약+링크)
- [ ] 검증 불가 항목은 상태 태그로 명확히 격리
- [ ] 변경 이력(일일 news + 실행 이력) 추적 가능

## 6) 주간 리뷰 출력(권장)
- [ ] Top 10 미검증 항목
- [ ] 신규 확정 항목
- [ ] 삭제/정정 항목(사유 포함)
- [ ] 다음 주 우선순위(작품/수상/인터뷰/화보)
