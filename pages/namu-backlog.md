# 나무위키 참고(2차) 보강 백로그

> 목적: 나무위키를 ‘누락 체크용 참고(2차)’로만 사용하여, 우리 위키를 **공식 출처(1차) 링크 기반**으로 점진적으로 완성합니다.
> 규칙: 항목을 추가할 때는 반드시 1차 출처 링크를 붙이고, `id:` + `sources/seen-urls.txt`로 중복 방지합니다.

## 진행 상태
- [ ] 광고/엠버서더 전체 확장 (MAA 프로필 AD 목록 + 나무위키 광고 목록)
- [ ] 화보/사진 전체 확장 (매거진 원문/공식 SNS)
- [ ] 방송/예능/행사/시상식 출연 기록 확장
- [ ] 뮤직비디오 출연 기록 정리(공식 MV 링크)
- [ ] 홍보대사/대외활동 정리(공식 위촉 기사/기관 발표)
- [ ] 수상/노미네이트 교차검증(시상식 공식 페이지)
- [ ] 작품별 페이지 링크 박스 보강(핵심 링크 1~5개로 압축)
- [ ] 스케줄(공식 공개 ‘미래 일정’) 채우기

## 오늘의 원칙(자동 업데이트 시)
- ‘많이 수집’은 하되, 최종 기록은 **공식 링크**로만 확정합니다.
- 공식 링크를 확보하지 못한 항목은 ‘후보/메모’로만 남기고 확정 페이지에는 넣지 않습니다.
