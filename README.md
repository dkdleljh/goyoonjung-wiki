# 고윤정 개인 위키 (Go Youn-jung)

배우 **고윤정**에 대한 정보를 **마크다운 파일 기반**으로 축적·정리하는 개인 위키입니다.

## 🚪 시작하기(포털)
- **덕질 허브(추천)** → [`pages/hub.md`](pages/hub.md)
- 메인 요약/바로가기 → [`index.md`](index.md)

## 🗓️ 매일 업데이트 확인
- 최근 7일 요약 → [`news/README.md`](news/README.md)
- 날짜별 로그 폴더 → [`news/`](news/)

## 🔎 출처/수집 출발점
- 고정 체크리스트(공식/신뢰 출처) → [`sources/watchlist.md`](sources/watchlist.md)
- 대량 수집용(매체별 내부검색 모음) → [`sources/media-watch.md`](sources/media-watch.md)
- 출처 목록 → [`sources/sources.md`](sources/sources.md)

## 📂 폴더 구조
- `pages/` : 고정 페이지(프로필/필모/수상/타임라인/아카이브/작품별 정리)
- `news/` : 날짜별 자동 업데이트 로그(매일 09:00)
- `sources/` : 출처 및 Watch 문서
- `scripts/` : 인덱스/점검/자동 후보 생성 스크립트
- `config/` : 자동 후보 키워드 등 설정

## ⚙️ 자동화
- 매일 **09:00(Asia/Seoul)** 위키 업데이트
- 매일 **09:10** 압축 백업 생성(`backups/`)
- 매일 **09:20** GitHub 자동 푸시

## 원칙
- 사생활/루머/추측은 기록하지 않습니다.
- 사실은 출처 링크와 함께 기록합니다.
- 이미지/화보는 **링크 중심**으로 기록합니다. (무단 저장·재배포 금지)
